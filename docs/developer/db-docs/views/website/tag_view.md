| Description |  |

**Source:**

```plpgsql
 CREATE OR REPLACE VIEW website.tag_view
 AS
 SELECT DISTINCT unnest(regexp_split_to_array(contents.tags, ','::text)) AS tag
   FROM website.contents;
```
